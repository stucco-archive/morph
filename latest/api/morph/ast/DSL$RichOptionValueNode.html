morph.ast.DSL

RichOptionValueNode

implicit class RichOptionValueNode extends AnyRef

The implicit class that provides the majority of the methods in the DSL.

These methods operate on Options because methods may or may not return results. By having these methods on Options, it is very easy to chain them and automatically filter out empty results in the end using the ^(...) object constructor or the *(...) array constructor.
